## 5. 输出报告接口（专项）

### 5.1 `/api/dividend/report/one-pager`（A4 一图版）

- **方法**: `GET`
- **响应**: `text/html;charset=utf-8`
- **Content-Disposition**: `attachment; filename*=UTF-8''dividend_report_{date}.html`
- **模板**：`backend/dividend-select/src/api/routes.py:_render_one_pager_html()`
- **数据**：`top_curr, top_3y, top_kofei, top_cagr, top_curr_bars, total_stocks, today_str`
- **数据源**：`_load_report_context()`（前后端共用的 4 个 TOP10 计算）

### 5.2 `/api/dividend/report/carousel`（手机竖版轮播）

- **方法**: `GET`
- **响应**: `text/html;charset=utf-8`
- **Content-Disposition**: `attachment; filename*=UTF-8''dividend_carousel_{date}.html`
- **模板**：`backend/dividend-select/src/api/routes.py:_render_carousel_html()`
- **数据**：复用 `_load_report_context()`
- **特色**：
  - 1080×1920 移动端竖版
  - iPhone 设备边框（黑色圆角 + 状态栏 + Dynamic Island + home indicator）
  - 4 slide 自动轮播（10s 间隔）
  - 每个 slide 含"⬇ 下载当前图"按钮（html2canvas → 1080×1920 PNG，含 iPhone 边框，文件名加 `_iphone` 后缀）

### 5.3 关键设计：实时价口径

两个报告 API 的"实时"列都用 **div / 实时价 × 100** 口径（**不是** CSV `2025年股息率(%)` 的年均价口径）。

实现：`routes.py:_load_report_context()` 内
```python
yield_realtime_map = {}
for _, _r in df.iterrows():
    _code = str(_r["股票代码"]).zfill(6)
    _div = _r.get("2025年分红(元/股)")
    _rt = m120_data.get(_code, {}).get("realtime")
    if pd.notna(_div) and _rt and _rt > 0:
        yield_realtime_map[_code] = round(float(_div) / float(_rt) * 100, 2)
df["_yield_realtime"] = df["股票代码"].astype(str).str.zfill(6).map(yield_realtime_map)
```

**结果**：华特达因 2.5/27.77 = 9.00%（与前端界面 9.00% 一致）；不再是旧版的 2.5/31.25 = 8.00%。
