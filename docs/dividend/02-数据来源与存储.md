## 2. 数据源清单

| # | 数据类型 | 数据源 | 调用位置（子模块） | CSV 存储 |

## 4. 更新频率建议

| 数据类型 | 建议频率 | 触发条件 |
|---------|---------|----------|
| 红利指数持仓 | **每季度** | 指数成分股调整 |
| 股票分红次数 | **每半年** | 历史数据变化慢 |
| 个股板块映射 | **每季度** | 板块归属变动 |
| **近 3 年股息率汇总** | **每月** | 新分红季（5-7月）需重算 |
| **M120 均线** | **每日** | 交易收盘后（15:30 后） |
| **PE/PB** | **每日** | 交易收盘后 |
| 实时股价 | **按需** | 用户查询时 |
| 股东户数 | **每季度** | 季报披露后 |
| 财务指标 | **每季度** | 季报披露后 |

## 5. 关键数据流：从 CSV 到前端

```
CSV (data/YYYY-MM/近3年股息率汇总.csv)
  ↓ data_reader.read_csv()
  ↓ 合并 stock_info_service.get_stocks_info() (sw_map 行业)
  ↓ 合并 m120_service.read_m120_with_deviation() (m120_data)
  ↓ 合并 financial_reader.read_csv() (financial_map 扣非/CAGR)
  ↓ 计算 yield_realtime_map = div / realtime × 100 (实时口径)
  ↓ df["_yield_realtime"] = ... (排序键)
  ↓ _load_report_context() 返回 {top_curr, top_3y, top_curr_bars, top_kofei, top_cagr, ...}
  ↓ _render_one_pager_html() / _render_carousel_html() 输出 HTML
  ↓ API /api/dividend/report/{one-pager,carousel}
  ↓ Next.js BFF /api/dividend/report/{one-pager,carousel}
  ↓ 前端 fetch + blob download
```

## 6. 阿里云行情 API 关键配置

- **Base URL**: `http://alirmcom2.market.alicloudapi.com`
- **认证**: Header `Authorization: APPCODE <key>`
- **限流保护**: 连续 5 次失败触发 `_rate_limited` 全局标志
- **⚠️ 凭证硬编码在 `core/calculator.py`**（CLAUDE.md 警告："不要在代码中硬编码凭证"——后续应挪到 `.env`）
