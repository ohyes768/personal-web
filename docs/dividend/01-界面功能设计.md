## 1. 模块概览

`apps/dividend` 是 A 股高股息率股票筛选模块，提供 4 大核心能力：

| # | 能力 | 说明 |
|---|------|------|
| 1 | **筛选** | 按交易所（沪市/深市主板）、最低股息率阈值过滤 |
| 2 | **TOP10 排序** | 按实时股息率 / 近 3 年均值 / 扣非同比 / 3 年 CAGR 4 个维度排名 |
| 3 | **详情查看** | 股票详情弹框（M120 偏离度、扣非同比、板块信息、分红历史） |
| 4 | **对比 + 报告** | 多股对比浮栏 + 输出 A4 一图版 / 手机竖版轮播 HTML |

技术栈：**Next.js 15 + React 19 + TypeScript + Tailwind CSS v4**

## 3. 核心功能详解

### 3.1 筛选与排序

- **交易所筛选**：下拉切换"全部 / 沪市主板 / 深市主板"
- **最低股息率阈值**：Header 输入框，默认 3.5%（可在 `apps/dividend/src/app/page.tsx` 调整 `minYieldInput` 默认值）
- **排序**：默认按"实时股息率"降序，可点击表头切换
  - 实时：来自 `tech.realtime` + `dividend_2025`（年均价口径或实时价口径，见下方 3.5 关键口径说明）
  - 近 3 年均值：来自 `dividend.avg_yield_3y`
  - 扣非同比 / 3 年 CAGR：来自 `financial.扣非净利润同比 / 3年复合增长率`

### 3.2 多股对比

- 每行右侧"对比"按钮（`+` 图标）→ 加入对比列表（最多 5 只）
- 底部 `CompareFloatingBar` 实时显示已选股
- 点击"对比"→ 打开 `CompareDrawer`（侧边抽屉）
- 抽屉内表格按"基础指标 / 价格 / 历史股息率"分组展示
- 支持 ESC 关闭、Tab 键盘导航（无障碍设计）

### 3.3 输出报告

Header 右侧 `输出报告 ▼` 下拉菜单，2 个选项：

| 选项 | 文件名 | 模板 | 特点 |
|------|--------|------|------|
| A4 一图版（横版） | `dividend_one_pager_YYYY-MM-DD.html` | 左右布局 2×2 | 适合电脑端分享/打印 |
| 手机竖版（轮播） | `dividend_carousel_YYYY-MM-DD.html` | 1080×1920 iPhone 模拟 + 4 slide 自动轮播 | 适合手机端/截图分享 |

每个 slide 含 ⬇ 下载原图按钮（PNG 1080×1920，含 iPhone 边框）。

### 3.4 数据刷新

Header 的 4 个刷新按钮：

| 按钮 | 调用 | 频率建议 |
|------|------|----------|
| 更新股息率 | `POST /api/dividend/refresh` | 每月/分红季 |
| 更新 M120 | `POST /api/m120/refresh` | 每日收盘后 |
| 更新实时价格 | `POST /api/realtime/refresh` | 按需 |
| 更新财务指标 | `POST /api/financial/refresh` | 季度 |

## 4. 关键设计决策

### 4.1 实时股息率口径（重要！）

**历史 bug**：报告下载的"实时"列用 `2025年股息率(%)`（CSV 字段，基于**年均价**），导致 8.00% 而非界面显示的 9.00%。

**修复（2026-06-02，commit `6de80bd` + `c77e05b`）**：
- 后端 `routes.py:_load_report_context` 新增 `yield_realtime_map = div / realtime × 100`
- 排序键从 `2025年股息率(%)` → `_yield_realtime`
- 6 处 `yield_curr` 取值改用 `_yield_realtime`
- **结果**：华特达因 8.00% → 9.00%，思维列控 9.47% → 11.63%，与前端界面完全一致

### 4.2 端口对齐

- 后端 dividend-select：`uvicorn --port 8099`（脚本里 `DIVIDEND_API_PORT=8099`）
- 前端 `page.tsx`：`fetch(\`${API_BASE}${endpoint}\`)`，`API_BASE = process.env.NEXT_PUBLIC_API_BASE_URL || 'http://localhost:8099'`
- Next.js BFF 代理路由（`apps/dividend/src/app/api/dividend/report/{one-pager,carousel}/route.ts`）**硬编码 `API_BASE = 'http://localhost:8080'`** ← ⚠️ 死代码，端口错位但实际不调用

## 6. 状态管理（React Hooks）

| Hook | 职责 | 数据源 |
|------|------|--------|
| `useDividendData()` | 股票列表 + 分红 + 板块 | `GET /api/dividend/stocks` |
| `useTechnicalData(codes)` | M120 偏离度 + 实时价 + 季度 TTM 股息率 | `GET /api/m120` + 实时数据 |
| `useDataUpdate()` | 4 个数据源刷新状态 | 4 个 `POST /api/.../refresh` |
| `useDetailModal()` | 详情弹框开关 + 当前股票 | 本地状态 |
| `useCompare(max)` | 对比列表 + 抽屉开关 | 本地状态 |

## 7. 关键 UI/UX 规范

- **响应式断点**：375px / 768px / 1024px / 1440px（详见技术规范）
- **可访问性**：ESC 关闭弹框、Tab 键盘导航、ARIA 属性、focus ring
- **动画**：150-300ms 过渡、`prefers-reduced-motion` 支持
- **iPhone carousel 特殊设计**：1080×1920 + 设备边框 + 状态栏 + Dynamic Island + home indicator + 4 slide 自动轮播（10s 间隔）+ html2canvas 截图含设备边框
