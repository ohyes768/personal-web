# 股息率展示前端界面 - 技术规范

## 1. 概述

### 1.1 背景

现有 A 股高股息率查询工具是一个纯 Python 后端项目，能够生成 `近3年股息率汇总.csv` 数据文件。用户需要通过 Web 界面可视化展示这些数据，方便筛选和查看股票的股息率信息。

### 1.2 目标

**核心目标：**
- 实现 Web 前端界面，展示近3年股息率数据
- 支持按股息率阈值筛选
- 支持按股息率排序
- 支持行展开查看季度股息率详情

**附加目标：**
- 使用虚拟滚动优化性能
- 使用 Docker Compose 统一部署

**非目标：**
- 不支持用户注册/登录
- 不支持数据修改
- 不支持实时数据推送（CSV更新后手动刷新）

## 2. 功能需求

### 2.1 核心功能

| 功能 | 描述 | 优先级 | 状态 |
|------|------|--------|------|
| 股息率阈值筛选 | 用户可自定义阈值，默认 3% | P0 | ✅ 已实现 |
| 交易所筛选 | 支持按交易所（上交所/深交所）筛选 | P1 | ✅ 已实现 |
| 数据表格展示 | 表格展示，支持页面滚动 | P0 | ✅ 已实现 |
| 详情弹框 | 季度详情、年度详情、板块、价格波动 | P0 | ✅ 已实现 |
| 申万行业显示 | 显示一、二、三级申万行业（拼接） | P1 | ✅ 已实现 |

### 2.2 用户故事

**场景1：筛选高股息率股票**
用户想要找到股息率高于 4% 的股票，输入阈值 4%，点击查询，表格自动更新显示符合条件的股票。

**场景2：按交易所筛选**
用户只想查看上交所的高股息率股票，选择交易所为「上交所」，点击查询进行筛选。

**场景3：查看季度详情**
用户点击某行股票的「季度详情」按钮，弹框显示该股票 2025 年 Q1-Q4 的季度股息率详情，了解分红的时间分布。

**场景4：查看价格波动**
用户点击「波动」按钮，查看该股票的价格区间和涨跌幅信息。

**场景5：查看板块信息**
用户点击「板块」按钮，查看该股票所属的概念板块和行业板块。

### 3.3 方案权衡记录

**页面滚动 vs 虚拟滚动 vs 表格内滚动**
- 考虑：数据量约几十到几百条，不需要虚拟滚动
- 选择：页面滚动
- 理由：代码更简洁、易维护，现代浏览器完全能流畅处理

**弹框 vs 展开行**
- 考虑：移动端和桌面端体验
- 选择：弹框
- 理由：弹框在移动端体验更好，不会影响表格布局

**横向对比 vs 纵向列表**
- 考虑：季度/年度数据的对比需求
- 选择：横向对比（指标为行，时间为列）
- 理由：横向对比更直观，方便看出趋势变化

**按钮触发 vs 自动防抖**
- 考虑：用户控制筛选时机
- 选择：按钮触发 + 回车触发
- 理由：避免不必要的请求，用户可主动控制查询时机

**卡片式 vs 表格式（波动详情）**
- 考虑：价格波动数据展示
- 选择：卡片式（2x2 价格卡片 + 2x2 涨跌卡片）
- 理由：简单直观，关键数字突出，颜色编码清晰（涨红跌绿）

### 4.1 数据结构

```typescript
interface DividendStock {

  sw_level2?: string | null;     // 申万二级行业
  sw_level3?: string | null;     // 申万三级行业

  avg_price_2025?: number | null;
  dividend_2025?: number | null;
  dividend_count_2025?: number | null;
  yield_2025?: number | null;

  avg_price_2024?: number | null;
  dividend_2024?: number | null;
  dividend_count_2024?: number | null;
  yield_2024?: number | null;

  // 2023 年数据
  avg_price_2023?: number | null;
  dividend_2023?: number | null;

  yield_2023?: number | null;

  // 3 年平均

  high_price_2025?: number | null;
```

### 5.2 交互设计

**表格交互：**
- 点击操作按钮：弹出对应详情弹框
- 表格跟随页面滚动，无独立滚动条

**筛选交互：**
- 输入阈值：回车或点击查询按钮触发筛选
- 选择交易所：点击查询按钮触发筛选
- 输入验证：最小值 0

**弹框交互：**
- 季度详情：横向对比显示 Q1-Q4 数据
- 年度详情：横向对比显示 2023-2025 数据
- 板块：标签式展示概念板块和行业板块
- 价格波动：卡片式展示价格信息和涨跌幅（涨红跌绿）

### 5.3 响应式设计

**桌面端：**
- 表格最大宽度 max-w-6xl
- 水平内边距 px-8 lg:px-16
- 首页卡片 2 列网格布局
- 表格横向滚动支持

## 7. 边缘情况与错误处理

### 7.1 异常场景

**网络异常：**
- 显示友好的错误提示
- 提供重试按钮

**数据异常：**
- 空值显示为 "-"
- 无效数据过滤

### 7.2 边界条件

**阈值输入：**
- 负数：自动修正为 0
- 超大值：限制最大值

**数据量为 0：**
- 显示暂无数据提示

### 7.3 降级策略

- API 不可用：显示缓存数据或友好提示
- CSV 文件缺失：返回 404 错误
