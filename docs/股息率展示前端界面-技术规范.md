## 1. 概述

### 1.1 背景

现有 A 股高股息率查询工具是一个纯 Python 后端项目，能够生成 `近3年股息率汇总.csv` 数据文件。用户需要通过 Web 界面可视化展示这些数据，方便筛选和查看股票的股息率信息。

### 1.2 目标

**核心目标：**
- 实现 Web 前端界面，展示近3年股息率数据
- 支持按股息率阈值筛选
- 支持按股息率排序
- 支持查看技术指标（PE/PB/M120）
- 支持实时价格刷新

**附加目标：**
- 使用虚拟滚动优化性能
- 使用 Docker Compose 统一部署

**非目标：**
- 不支持用户注册/登录
- 不支持数据修改
- 不支持实时数据推送（CSV更新后手动刷新）

### 2.2 用户故事

**场景1：筛选高股息率股票**
用户想要找到股息率高于 4% 的股票，输入阈值 4%，点击查询，表格自动更新显示符合条件的股票。

**场景2：查看技术指标**
用户在表格中可以快速查看每只股票的 PE、PB、M120、昨日收盘价等关键指标，评估投资价值。

**场景3：刷新实时价格**
用户点击某只股票的刷新按钮，获取该股票的实时最新价格和偏离度，了解当前价格相对于 M120 的位置。

**场景4：查看季度详情**
用户点击某行股票的「季度详情」按钮，弹框显示该股票 2025 年 Q4-Q1 的季度股息率详情（按时间倒序）。

**场景5：查看板块信息**
用户点击「板块」按钮，查看该股票所属的概念板块和行业板块。

## 3. 技术决策

### 3.1 技术栈

**模块划分：**

- `types/` - TypeScript 类型

### 3.3 方案权衡记录

**实时价格刷新方式**
- 考虑：轮询 vs 手动刷新
- 选择：手动刷新（单只股票）
- 理由：减少 API 调用，避免性能开销

**akshare 接口选择**
- 考虑：全量数据 vs 单只股票接口
- 选择：`stock_intraday_em` 单只股票分时接口
- 理由：只获取单只股票，响应速度快

**表格列布局**
- 考虑：横向扩展 vs 垂向堆叠
- 选择：部分字段采用垂向堆叠（如 PE/PB、申万行业）
- 理由：节省横向空间，提高数据密度

**季度显示顺序**
- 考虑：Q1-Q4 vs Q4-Q1
- 选择：Q4-Q1（时间倒序）
- 理由：最新的季度在前，符合用户浏览习惯

#### 季度数据模型

```typescript

  q4: { avg_price?: number | null; dividend?: number | null; yield_pct?: number | null; };

  q1: { avg_price?: number | null; dividend?: number | null; yield_pct?: number | null; };

```

#### 技术指标数据模型

```typescript
interface TechnicalIndicators {
  pe?: number | null;           // 市盈率
  pb?: number | null;           // 市净率
  m120?: number | null;         // 120日均线
  close?: number | null;        // 昨日收盘价（从 M120 数据获取）
  deviation?: number | null;    // 与 M120 的偏离度（百分比）
  realtimeClose?: number | null; // 实时价格（刷新后更新）
  realtimeDeviation?: number | null; // 实时偏离度（刷新后更新）
}
```

#### 实时价格请求/响应模型

```typescript
interface RealtimePriceRequest {
  code: string;      // 股票代码
  m120: number;      // M120 值，用于计算偏离度
}

interface RealtimePriceResponse {
  code: string;
  close?: number | null;       // 实时价格
  deviation?: number | null;   // 与 M120 的偏离度
  timestamp?: string | null;   // 时间戳
}
```

### 5.2 交互设计

**表格交互：**
- 点击操作按钮：弹出对应详情弹框
- 表格跟随页面滚动，无独立滚动条
- 点击刷新图标：获取单只股票实时价格

**筛选交互：**
- 输入阈值：回车或点击查询按钮触发筛选
- 选择交易所：点击查询按钮触发筛选
- 输入验证：最小值 0

**弹框交互：**
- 季度详情：横向对比显示 Q4-Q1 数据（按时间倒序）
- 年度详情：横向对比显示 2023-2025 数据
- 板块：标签式展示概念板块和行业板块
- 价格波动：卡片式展示价格信息和涨跌幅（涨红跌绿）

**刷新交互：**
- 点击刷新图标：请求实时价格
- 加载中：显示旋转图标
- 成功后：显示实时价格和偏离度
- 失败时：显示错误提示
- 缓存机制：1 天内复用缓存

### 5.3 响应式设计

**桌面端：**
- 表格最大宽度 max-w-6xl
- 水平内边距 px-8 lg:px-16

## 7. 边缘情况与错误处理

### 7.1 异常场景

**网络异常：**
- 显示友好的错误提示
- 提供重试按钮

**数据异常：**
- 空值显示为 "-"
- 无效数据过滤

### 7.2 边界条件

**阈值输入：**
- 负数：自动修正为 0
- 超大值：限制最大值

**数据量为 0：**
- 显示暂无数据提示

### 7.3 降级策略

- API 不可用：显示缓存数据或友好提示
- CSV 文件缺失：返回 404 错误
