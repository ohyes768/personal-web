## 技术栈

### 前端

- **框架**: Next.js 15.4 + React 19
- **样式**: Tailwind CSS v4
- **语言**: TypeScript
- **图表**: Recharts

### 后端

- **API Gateway**: FastAPI (Python 3.11)
- **微服务**:
  - FinancialDashboard (端口 8091)
  - xwlb-analyze (端口 8092)
  - douyin-collect (端口 8093) - 视频数据采集
  - file-system-go (端口 8095) - 文件存储和元数据服务
- **子模块**:
  - douyin-processor (端口 8094) - ASR 音频识别处理 (独立仓库)

### 部署

- **容器**: Docker + Docker Compose
- **反向代理**: Nginx
- **服务器**: 阿里云 ECS
