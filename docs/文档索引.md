## 技术栈

### 前端

- **框架**: Next.js 15.4 + React 19
- **样式**: Tailwind CSS v4
- **语言**: TypeScript
- **图表**: Recharts

### 后端

- **API Gateway**: FastAPI (Python 3.11)
- **微服务**: 
  - FinancialDashboard (端口 8091)
  - xwlb-analyze (端口 8092)
  - douying-collect (端口 8093)

### 部署

- **容器**: Docker + Docker Compose
- **反向代理**: Nginx
- **服务器**: 阿里云 ECS

## 常见问题

### Q1: 如何快速部署到阿里云？

**A**: 参考 [DOCKER_DEPLOY.md](../DOCKER_DEPLOY.md) 快速部署指南，3 步完成部署。

### Q2: 如何查看服务日志？

**A**: 
```bash
# 查看所有服务日志
docker compose -f docker-compose.prod.yml logs -f

# 查看特定服务日志
docker compose -f docker-compose.prod.yml logs -f gateway
```

### Q3: 如何备份数据？
