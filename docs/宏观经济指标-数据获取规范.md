# 宏观经济指标数据获取规范

## 概述

本文档说明项目中所有宏观经济指标的数据来源、获取方式和使用方法。

---

## 数据源总览

| 数据源 | 类型 | 指标数量 | API Key |
|--------|------|----------|---------|
| FRED API | 利率、汇率、VIX | 10+ | 需注册 |
| ECB API | 欧元区国债 | 3 | 免费 |
| AKShare | 资金流向、国债收益率、货币供应、社融、指数 | 10+ | 免费 |

## FRED API 指标

### 配置

**环境变量**：
```
FRED_API_KEY=your_api_key_here
```

**API Key 获取**：https://fred.stlouisfed.org/docs/api/api_key.html

### 利率指标

| 指标 | 代码 | 说明 | 频率 |
|------|------|------|------|
| 3M 美债 | DGS3MO | 3个月国债收益率 | 日 |
| 2Y 美债 | DGS2 | 2年期国债收益率 | 日 |
| 10Y 美债 | DGS10 | 10年期国债收益率 | 日 |
| SOFR | SOFR | 隔夜担保融资利率 | 日 |
| IORB | IORB | 准备金余额利率（政策利率） | 日 |

### 汇率指标

| 指标 | 代码 | 说明 |
|------|------|------|
| 美元指数 | DTWEXBGS | 美元指数 |
| USD/CNY | DEXCHUS | 美元兑人民币 |
| USD/JPY | DEXJPUS | 美元兑日元 |
| USD/EUR | DEXUSEU | 美元兑欧元（需取倒数） |

### VIX 指标

| 指标 | 代码 | 说明 |
|------|------|------|
| VIX | VIXCLS | CBOE 波动率指数 |

### 欧/日利率（OECD 数据）

| 指标 | 代码 | 说明 | 频率 |
|------|------|------|------|
| 德国3M银行利率 | IR3TIB01DEM156N | 3个月银行间利率 | 月 |
| 德国10Y国债 | IRLTLT01DEM156N | 10年期国债收益率 | 月 |
| 日本3M银行利率 | IR3TIB01JPM156N | 3个月银行间利率 | 月 |
| 日本10Y国债 | IRLTLT01JPM156N | 10年期国债收益率 | 月 |

### 获取方式

**Python 代码**：
```python
from fredapi import Fred

fred = Fred(api_key=os.getenv('FRED_API_KEY'))
data = fred.get_series('SERIES_CODE', observation_start='2025-01-01')
```

**API 调用**：
```
https://api.stlouisfed.org/fred/series/observations?series_id=SERIES_CODE&api_key=YOUR_KEY
```

## ECB API 指标

### 德国国债（欧元区数据）

| 指标 | 代码 | 说明 |
|------|------|------|
| 2Y 德债 | M.U2.EUR.4F.BB.U2_2Y.YLD | 2年期 |
| 5Y 德债 | M.U2.EUR.4F.BB.U2_5Y.YLD | 5年期 |
| 10Y 德债 | M.U2.EUR.4F.BB.U2_10Y.YLD | 10年期 |

### 获取方式

**API 端点**：
```
https://data-api.ecb.europa.eu/service/data/FM/SERIES_CODE
```

**代码示例**：
```python
import requests

url = f"https://data-api.ecb.europa.eu/service/data/FM/M.U2.EUR.4F.BB.U2_10Y.YLD"
params = {
    "startPeriod": "2025-01-01",
    "endPeriod": "2025-12-31",
    "format": "jsondata"
}
response = requests.get(url, params=params)
```

| 指标 | AKShare 接口 | 代码 | 说明 |
|------|-------------|------|------|
| 上证指数 | stock_zh_index_daily_tx | sh000001 | ✅ 可用 |
| 中证全指（全A替代） | stock_zh_index_daily_tx | sh000985 | ✅ 可用，可作为全A指数替代 |
| 万得全A指数 | - | 881001 | ❌ 需万得付费 |

**说明**：
- 万得全A指数属于万得付费数据，AKShare 无免费接口
- 中证全指（sh000985）包含沪深两市全部股票，可作为全A市场表现的替代指标

### 获取方式

**资金流向**：
```python
import akshare as ak

df = ak.stock_market_fund_flow()
# 列: 日期, 上证净流入, 深证净流入, 北向净流入, 南向净流入等
```

**中国国债收益率（2年期）**：
```python
import akshare as ak

# 获取中国国债收益率数据（包含2年期）
df = ak.bond_zh_us_rate()

# 筛选最近数据
df['日期'] = pd.to_datetime(df['日期'])
df_recent = df[df['日期'] >= '2025-01-01']

# 获取2年期国债收益率
bond_2y = df_recent[['日期', '中国国债收益率2年']]
```

**中国国债收益率（其他期限）**：
```python
import akshare as ak

# 获取国债收益率数据
df = ak.bond_china_yield(start_date='20250101', end_date='20250311')

# 筛选中债国债收益率曲线
bond_curve = df[df['曲线名称'] == '中债国债收益率曲线']

# 获取10年期国债收益率
bond_10y = bond_curve[['日期', '10年']]
```

**货币供应量**：
```python
import akshare as ak

# 获取 M1、M2 增速
df = ak.macro_china_money_supply()
# 列: 月份, M2金额, M2同比, M2环比, M1金额, M1同比, M1环比, M0金额, M0同比, M0环比

# 计算 M1-M2 剪刀差
df['M1_M2剪刀差'] = df['M1同比'] - df['M2同比']
```

**社会融资数据**：
```python
import akshare as ak

# 获取社融数据
df = ak.macro_china_new_financial_credit()
# 列: 月份, 新增, 新增同比, 新增环比, 累计, 累计同比
```

**股票指数**：
```python
import akshare as ak

# 上证指数
df_sh = ak.stock_zh_index_daily_tx(symbol='sh000001')
# 列: ['date', 'open', 'close', 'high', 'low', 'amount']

# 中证全指（全A替代）
df_za = ak.stock_zh_index_daily_tx(symbol='sh000985')
# 列: ['date', 'open', 'close', 'high', 'low', 'amount']
```

## 可计算利差指标

### SOFR-IORB 利差

| 属性 | 值 |
|------|-----|
| 计算 | SOFR - IORB |
| 含义 | 市场利率 vs 政策利率 |
| 用途 | 流动性监控、政策传导效果 |

### SOFR-DGS3MO 利差

| 属性 | 值 |
|------|-----|
| 计算 | SOFR - DGS3MO |
| 含义 | 市场利率 vs 国债利率 |
| 用途 | TED Spread 替代、市场压力监测 |

### 期限结构利差（美国）

| 指标 | 计算 |
|------|------|
| 2Y-3M | DGS2 - DGS3MO |
| 10Y-2Y | DGS10 - DGS2 |
| 10Y-3M | DGS10 - DGS3MO |

### 期限结构利差（中国）

| 指标 | 计算 | 数据来源 |
|------|------|----------|
| 2Y-1Y | 2年 - 1年 | AKShare bond_zh_us_rate / bond_china_yield |
| 10Y-2Y | 10年 - 2年 | AKShare bond_zh_us_rate / bond_china_yield |
| 10Y-5Y | 10年 - 5年 | AKShare bond_china_yield |
