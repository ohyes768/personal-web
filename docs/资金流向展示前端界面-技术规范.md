# 资金流向展示前端界面 - 技术规范

## 1. 概述

### 1.1 背景
全球宏观经济金融数据服务中新增资金流向模块，展示沪深港通北向/南向资金流向数据及累计指标。

### 1.2 目标
- 核心目标：在宏观经济数据页面新增"资金流向"tab，提供专业化的资金流向可视化展示
- 非目标：移动端适配

## 4. 组件说明

### 4.1 FundFlowChart.tsx（图表组件）
**功能**：使用 TradingView Lightweight Charts 展示资金流向趋势图

**主要特性**：
- 创建三条 `LineSeries`：北向资金（绿色）、南向资金（红色）、净流入（蓝色）
- 支持时间范围过滤（1M/3M/6M/1Y/ALL）
- 响应式窗口大小调整
- 十字准星悬停提示

**关键代码**：
```typescript
import { createChart, LineSeries } from 'lightweight-charts';

// 创建图表
const chart = createChart(container, {
  width: container.clientWidth,
  height: 500,
  layout: { background: { color: '#131722' } },
  // ...
});

// 创建曲线
const seriesNorth = chart.addSeries(LineSeries, {
  color: '#26a69a',
  lineWidth: 2,
  title: '北向',
});
```

### 4.2 IndicatorCards.tsx（指标卡片）
**功能**：展示北向/南向/净流入的 7D/30D 累计数据

**主要特性**：
- 三列网格布局
- 正负值颜色区分（绿/红）
- 金额格式化（保留 2 位小数）

### 4.3 TimeRangeSelector.tsx（时间范围选择器）
**功能**：提供时间范围快速切换

**主要特性**：
- 5 个预设按钮：1M、3M、6M、1Y、ALL
- 激活状态高亮（蓝色背景）

## 5. API 接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `http://localhost:8094/api/fund-flow/history` | GET | 获取历史数据 |
| `http://localhost:8094/api/fund-flow/cumulative` | GET | 获取累计数据（7D/30D） |

### 5.1 历史数据响应
```json
{
  "data": [
    {
      "date": "2025-09-05",
      "north_net": 415.99,
      "south_net": 11.84,
      ...
    }
  ]
}
```

### 5.2 累计数据响应
```json
{
  "north_cumulative": {
    "date": "2025-09-05",
    "cum_7d": 123.45,
    "cum_30d": 456.78
  },
  "south_cumulative": {
    "date": "2025-09-05",
    "cum_7d": 23.45,
    "cum_30d": 56.78
  }
}
```

## 7. 类型定义

```typescript
// 时间范围类型
export type TimeRange = '1M' | '3M' | '6M' | '1Y' | 'ALL';

// 累计数据
export interface CumulativeData {
  north_cumulative: CumulativeItem;
  south_cumulative: CumulativeItem;
}

export interface CumulativeItem {
  date: string;
  cum_7d?: number;
  cum_30d?: number;
}

// 历史数据
export interface HistoryDataItem {
  date: string;
  north_net?: number;
  north_buy?: number;
  north_sell?: number;
  south_net?: number;
  south_buy?: number;
  south_sell?: number;
}

// 图表数据
export interface ChartData {
  time: number;          // Unix timestamp (秒)
  northNet: number;
  southNet: number;
  netFlow: number;
}
```
