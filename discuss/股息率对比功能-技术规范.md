# 股息率筛选页面对比功能 - 技术规范 v2.0

## 1. 概述

### 1.1 背景

股息率筛选页面已具备基本的股票列表展示、筛选和详情查看功能。为了提升用户体验，需要添加股票对比功能，允许用户同时对比多只股票的关键指标，便于做出投资决策。

### 1.2 目标

- 允许用户勾选最多5只股票进行对比
- 在侧边抽屉中横向展示对比数据
- 支持多种排序方式和指标高亮
- 提供流畅的动画和交互体验
- 确保移动端和桌面端的一致体验
- 满足无障碍访问标准（WCAG 2.1 AA）

### 1.3 非目标

- 不支持自定义对比指标
- 不支持导出对比数据
- 不支持保存对比结果

### 1.4 设计系统

```
风格：Dark Data Dashboard
配色：Slate/Zinc 体系 + 功能色（蓝/绿/红）
字体：Inter（英文）+ 等宽数字字体
圆角：8px（按钮/卡片）/ 4px（输入框）
阴影：subtle，避免过重
动画：300ms ease-out，检查 reduced-motion
```

## 2. 功能需求

### 2.1 核心功能

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 勾选股票 | 在操作列点击"对比"按钮切换选中状态 | P0 |
| 底部浮动栏 | 显示已选数量、股票名称、进度指示、清空/开始对比 | P0 |
| 侧边抽屉 | 从右侧滑出，展示对比表格，支持固定表头 | P0 |
| 排序切换 | 支持按股息率/PE/偏离度/勾选顺序排序 | P1 |
| 指标高亮 | 高亮最优指标（股息率最高、PE最低、偏离度最大）+ 图标标记 | P1 |
| 移除股票 | 在对比窗口内移除不需要的股票 | P0 |
| 键盘导航 | ESC关闭抽屉，Tab顺序符合视觉顺序 | P0 |

### 2.2 用户故事

1. 用户想要对比3只股票的股息率、PE、PB等指标
2. 用户勾选3只股票后，底部浮动栏显示已选股票名称和进度
3. 用户点击"开始对比"打开对比窗口，表格带固定表头
4. 用户在对比窗口中切换排序方式，快速找到最优选择
5. 用户点击移除按钮，股票淡出消失
6. 用户点击"清空"按钮，重新选择其他股票对比
7. 用户按ESC键快速关闭抽屉

### 4.2 排序逻辑

```typescript
function sortStocks(stocks: CompareStockData[], sortBy: CompareSortType): CompareStockData[] {
  switch (sortBy) {
    case 'yield':
      return [...stocks].sort((a, b) => (b.avg_yield_3y ?? 0) - (a.avg_yield_3y ?? 0));
    case 'pe':
      return [...stocks].sort((a, b) => (a.technical?.pe ?? Infinity) - (b.technical?.pe ?? Infinity));
    case 'deviation':
      return [...stocks].sort((a, b) =>
        Math.abs(b.technical?.deviation ?? 0) - Math.abs(a.technical?.deviation ?? 0)
      );
    case 'selection':
      return [...stocks].sort((a, b) => a.displayOrder - b.displayOrder);
  }
}
```

### 4.3 高亮计算逻辑

```typescript
interface HighlightInfo {
  yieldIndex: number | null;    // 最高股息率
  peIndex: number | null;       // 最低 PE
  pbIndex: number | null;       // 最低 PB
  deviationIndex: number | null; // 最大偏离度
}

function calculateHighlights(stocks: CompareStockData[]): HighlightInfo {
  const yields = stocks.map(s => s.avg_yield_3y ?? -Infinity);
  const pes = stocks.map(s => s.technical?.pe ?? Infinity);
  const pbs = stocks.map(s => s.technical?.pb ?? Infinity);
  const deviations = stocks.map(s => Math.abs(s.technical?.deviation ?? 0));

  return {
    yieldIndex: yields.indexOf(Math.max(...yields)),
    peIndex: pes.indexOf(Math.min(...pes)),
    pbIndex: pbs.indexOf(Math.min(...pbs)),
    deviationIndex: deviations.indexOf(Math.max(...deviations)),
  };
}
```

```tsx
)}
```

### 8.2 空状态

```tsx
{stocks.length === 0 ? (
  <div className="text-center py-16">
    <EmptyStateIcon className="w-16 h-16 mx-auto text-gray-600 mb-4" />
    <h3 className="text-lg font-semibold text-gray-400 mb-2">
      暂无对比数据
    </h3>
    <p className="text-sm text-gray-500">
      请从列表中选择 2-5 只股票进行对比
    </p>
  </div>

  <CompareTable stocks={stocks} />
)}
```

### 8.3 撤销功能

```tsx
{showUndo && (
  <div className="fixed bottom-20 left-1/2 -translate-x-1/2 z-50">
    <div className="bg-gray-800 rounded-lg px-4 py-3 flex items-center gap-3 shadow-lg border border-gray-700">
      <span className="text-sm text-gray-300">

```

## 9. 边缘情况与错误处理

### 9.1 异常场景

| 场景 | 处理方式 | 优先级 |
|------|----------|--------|
| 少于2只对比 | Toast 提示"请至少选择 2 只股票进行对比" | P0 |
| 超过5只勾选 | 按钮禁用 + tooltip 提示 | P0 |
| 空数据 | 显示 "-" 占位 | P0 |
| 网络错误 | 错误提示 + 重试按钮 | P0 |
| 数据加载中 | Loading 动画 | P0 |
| 抽屉内容为空 | 空状态提示 | P1 |

### 9.2 边界条件

- 选中数量范围：0-5
- 比较数量范围：2-5
- 高亮计算：至少有1个非空值
- 响应式断点：xs/sm/md/lg/xl

### 9.3 降级策略

- 动画不支持：直接显示最终状态
- CSS grid不支持：使用 flex 布局
- Portal不支持：直接在组件内渲染
- reduced-motion：禁用所有动画
