# 股息率展示前端界面 - 技术规范（v2.0）

> 基于深度访谈生成的详细技术规范
> 生成时间: 2026-03-11

---

## 1. 概述

### 1.1 背景

- 现有 `dividend-select` 后端已实现 FastAPI 服务，提供股息率数据 API
- 用户需要 Web 前端界面可视化展示这些数据
- 需遵循 `前端模块化开发规范.md` 进行开发

### 1.2 目标

**核心目标：**
- 实现 Web 前端界面，展示近 3 年股息率数据
- 支持按股息率阈值筛选（默认 3%）
- 支持按股息率排序（默认降序）
- 通过弹框查看季度详情、板块/行业、年度详情

**附加目标：**
- 使用虚拟滚动优化性能（固定行高）
- 使用 Docker Compose 统一部署

**非目标：**
- 不支持用户注册/登录
- 不支持数据修改
- 不支持实时数据推送

## 2. 功能需求

### 2.1 核心功能

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 股息率阈值筛选 | 用户可自定义阈值，默认 3%，失焦/回车触发 | P0 |
| 数据表格展示 | 虚拟滚动（固定行高），显示最小化列 | P0 |
| 按股息率排序 | 默认 3 年平均股息率降序，支持升序/降序切换 | P0 |
| 详情弹框 | 三个独立弹框：季度详情、板块/行业、年度详情 | P0 |

### 2.2 用户故事

**场景1：筛选高股息率股票**
用户想要找到股息率高于 4% 的股票，输入阈值 4%，按回车键或失焦，表格自动更新显示符合条件的股票。

**场景2：查看季度详情**
用户点击某行股票的「季度详情」按钮，弹出弹框显示该股票 2025 年 Q1-Q4 的季度股息率详情表格。

**场景3：查看板块/行业**
用户点击某行股票的「板块/行业」按钮，弹出弹框以标签云形式展示该股票的概念板块和行业板块。

**场景4：查看年度详情**
用户点击某行股票的「年度详情」按钮，弹出弹框以表格形式展示该股票 2025/2024/2023 的分红详情。

### 3.3 方案权衡记录

**虚拟滚动 vs 分页**
- 选择：虚拟滚动（固定行高）
- 理由：用户可以快速浏览全部数据，无需翻页，体验更流畅

**后端排序 vs 前端排序**
- 选择：后端排序
- 理由：数据源统一，未来数据量增长时性能更好

**手动刷新 vs 自动更新**
- 选择：手动刷新（浏览器刷新）
- 理由：实现简单，避免轮询带来的服务器压力

**弹框 vs 行展开**
- 选择：弹框
- 理由：用户访谈确认，每行三个独立按钮点击弹框

**共享 Modal vs 独立 Modal**
- 选择：共享 Modal 组件
- 理由：减少重复代码，便于统一管理样式和行为

```typescript
  dividend_count_2025?: number | null; // 2025年分红次数

  // 2024 年数据
  avg_price_2024?: number | null;
  dividend_2024?: number | null;
  dividend_count_2024?: number | null;
  yield_2024?: number | null;

  // 2023 年数据
  avg_price_2023?: number | null;
  dividend_2023?: number | null;
  dividend_count_2023?: number | null;
  yield_2023?: number | null;

  // 3 年平均
  avg_price_3y?: number | null;   // 近3年平均股价
  avg_yield_3y?: number | null;   // 3年平均股息率(%)

/**
 * 季度数据模型
 */
interface QuarterlyData {
  q1?: Quarter | null;
  q2?: Quarter | null;

interface Quarter {

}
```

## 7. 边缘情况与错误处理

### 7.1 异常场景

**网络异常：**
- 显示骨架屏（加载中）
- 错误时显示友好提示 + 重试按钮

**数据异常：**
- 空值统一显示为 "-"
- 无效数据过滤

### 7.2 边界条件

**阈值输入：**
- 负数：自动修正为 0
- 超大值：不限制

**数据量为 0：**
- 显示「暂无数据」友好提示

**并发请求：**
- 防抖处理（500ms）

### 7.3 降级策略

- API 不可用：显示友好提示 + 重试按钮
- CSV 文件缺失：后端返回 404，前端显示错误

```
前端组件 → 模块 Hooks → 模块 API → 统一客户端 → API 路由代理 → 后端 API
```

**API 路由代理：**
```
frontend/src/app/api/dividend/[...path]/route.ts
→ 将请求代理到后端 FastAPI 服务
→ 解决跨域问题
```

### 10.5 术语表

| 术语 | 说明 |
|------|------|
| 虚拟滚动 | 只渲染可视区域内的数据行，提升大量数据渲染性能 |
| 股息率 | 每股分红 / 股价 × 100% |
| 季度股息率 | 按季度计算的股息率 |
| Portal | React Portal，将子节点渲染到父组件 DOM 层级之外 |
| 防抖 | 延迟执行函数，在延迟时间内再次触发则取消前一次 |
| Error Boundary | React 错误边界，捕获子组件树中的 JavaScript 错误 |
